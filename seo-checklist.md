## ✅ 5. Review Strategy – Recensioni Google

> Le recensioni sono il **fattore #1 per il local pack** di Google.

- [ ] Crea il tuo **link diretto alle recensioni Google** (da GBP → "Chiedi recensioni")
- [ ] Stampa un cartellino/QR code in negozio che porta alla pagina recensioni
- [ ] Dopo ogni servizio completato, chiedi verbalmente al cliente una recensione
- [ ] Invia follow-up via WhatsApp con link diretto
- [ ] **Rispondi a TUTTE le recensioni** (positive e negative) entro 24-48h
- [ ] Obiettivo minimo: **20+ recensioni** con media ≥ 4.5 stelle entro 3 mesi
- [ ] Chiedi recensioni anche su Yelp e Pagine Gialle
